# REACT NATIVE

## Prerequisites
* `NodeJS`
* `npm`
* `yarn`
* `react-native-cli`
* `XCode` v13.2.1 or higher
* `Android Studio v11.0.11 or higher`

## Setup
* `yarn global add expo-cli`
* `yarn`
* `cd ios && pod install`
* `yarn start`

## Run on iOS
* `react-native run-ios`
## Run on Android emulator
* `react-native run-android`
